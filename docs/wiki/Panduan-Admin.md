# 🛠️ Panduan Administrator

Sebagai Superadmin, Anda memiliki kontrol penuh atas integritas data sistem.

## Manajemen Pegawai
1.  **Tambah Pegawai**: Masuk ke menu **Pegawai > Tambah**. Isi NIP, Nama, Jabatan, dan Unit Kerja.
2.  **Bulk Import**: Gunakan fitur **Impor Excel** untuk memasukkan ratusan data pegawai sekaligus menggunakan template yang disediakan.
3.  **Ekspor Data**: Anda bisa mengunduh rekap data pegawai dalam format **PDF** atau **Excel** untuk pelaporan fisik.

## Verifikasi Dokumen
Ini adalah inti dari kontrol kualitas data:
1.  **Antrian Verifikasi**: Cek widget **Antrian Verifikasi** di Dashboard.
2.  **Review Dokumen**: Buka dokumen, periksa keaslian file.
3.  **Tindakan**:
    - **Verify**: Tandai dokumen sebagai valid.
    - **Reject**: Tolak dokumen dengan memberikan alasan yang jelas (misal: "File buram" atau "Tanda tangan salah").
    - **Lock**: Gunakan fitur **Kunci Dokumen** untuk mencegah pegawai mengubah file yang sudah diverifikasi secara permanen.

## Keamanan & Audit
- **Audit Logs**: Pantau siapa melakukan apa di sistem melalui menu **Audit Trail**. Setiap aktivitas (Login, Hapus Data, Ganti Foto) terekam secara permanen.
- **Pengaturan Sistem**: Kelola daftar **Unit Kerja** dan **Jabatan** agar tetap relevan dengan struktur organisasi terbaru.
